- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

# 🎓 Favourite Student List

A simple and interactive **Favourite Student List** application built with **React JS**, **React Router**, **Context API (useContext)**, and **Tailwind CSS**. This project demonstrates client-side routing, global state management, and dynamic UI updates by allowing users to add and remove favourite students across multiple pages.

## 🚀 Features

* 📋 Display a list of students dynamically using `map()`
* ⭐ Add students to the favourite list
* 🚫 Prevent duplicate students from being added
* ❌ Remove students from the favourite list
* 🌐 Global state management using **Context API** (`createContext` & `useContext`)
* 🔄 Client-side navigation with **React Router**
* ⚡ Instant UI updates without page reload
* 💬 Conditional rendering to display **"No favourite students added yet"** when the list is empty
* 🎨 Responsive and clean user interface built with **Tailwind CSS**

## 🛠️ Technologies Used

* React JS
* React Router
* Context API (`createContext`, `useContext`)
* JavaScript (ES6+)
* Tailwind CSS

## 📚 Learning Outcomes

This project helped in understanding:

* React functional components
* React Router for multi-page navigation
* Global state management using Context API
* Sharing state across components with `useContext`
* Rendering lists with `map()`
* Conditional rendering in React
* Preventing duplicate data entries
* Building responsive UIs with Tailwind CSS
